# WE Learn 自动化答题辅助脚本 (WE-Learn-Answer-Automation)

一个运行在浏览器控制台（Console）的轻量级前端自动化答题辅助脚本，专为 WE Learn 平台设计。通过底层 DOM 解析技术，自动提取并模拟填充页面内隐藏的正确答案。

## 💡 当前版本设计哲学：单任务可控版

本版本为 **单任务逐一手动触发版本**。相比于“一键全做完”的激进方案，该版本拥有更高的**安全性**与**自由度**：
- **自主掌控进度：** 允许使用者自己控制页面停留时间，自由掌握任务节奏。
- **降低风控风险：** 避免了由于瞬间完成整页题目而被平台后台策略检测为异常请求的风险。
- **查漏补缺：** 脚本执行后会对正确答案进行**绿色高亮视觉标记**，方便在提交前进行人工核对。

## 🚀 已支持题型

脚本通过智能识别 HTML 标签，目前已完美支持以下常见题型：
- **单项选择题 (`<li>`)**：自动模拟点击正确选项并高亮。
- **单空填空题 (`<input>`)**：自动注入标准答案，并触发系统输入事件。
- **长文本答题/翻译题 (`<textarea>`)**：自动填充大段文本，防止格式错乱。

## ⚠️ 局限性说明

- **暂不支持选段题**：由于选段题（如段落匹配、多选拖拽等）在前端的 DOM 结构较为复杂，**目前版本暂时无法处理**。遇到此类题型脚本会自动安全跳过，不会引发页面崩溃。

## 🛠️ 使用方法（新手必看避坑指南）

为了确保脚本能顺利读取到题目数据，请严格按照以下步骤操作：

1. **激活题目区域（关键步骤）**：
   进入 WE Learn 对应的答题页面后，先在**左侧题目区域**的任意文字或选项上**鼠标右键**，选择 **“检查”**（Inspect）。此操作可以帮控制台精准定位当前的题目作用域，防止脚本找不到元素。

2. **打开控制台**：
   键盘按下 `F12`（或在刚才打开的右侧/下方开发者工具面板中），切换到 **Console（控制台）** 标签页。

3. **解除安全粘贴限制（仅首次使用需要）**：
   如果是第一次在该浏览器使用脚本，直接粘贴代码可能会被浏览器拦截并警告。此时：
   - 请先在控制台输入框中手动输入：`允许粘贴` （部分英文系统浏览器请捕捉提示输入：`allow pasting`），然后按下回车。

4. **运行脚本**：
   将精修后的 JavaScript 脚本代码整体复制，粘贴到控制台中，按下 **Enter（回车）** 键，即可看到全自动化填充效果及控制台汇总日志。
